# KNUST Digital Academic Library Ecosystem

## Overview
The Digital Academic Library Ecosystem is a software platform that helps KNUST students organize their academic and personal lives through learning resources, AI-powered study tools, and smart scheduling.

## Features
- Ocean of PDFs
- AI Study Pavilion
- Interactive Smart Schedules
- Lecturer Resource Management
- Past Questions Repository
- AI Chatbot Tutor
- Flashcard Generation
