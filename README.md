# 📚 KNUST Digital Academic Library Ecosystem

---

## 🌍 Overview
The **Digital Academic Library Ecosystem** is a unified academic platform designed for KNUST students and lecturers to manage learning resources, schedules, and academic progress efficiently.

It combines **structured digital libraries**, **AI-assisted learning**, and **smart scheduling** into one powerful system.

---

## 🚀 Key Features

### 📖 Ocean of PDFs
- Organized academic materials by year, department, and course code  
- Lecturer-uploaded notes, textbooks, and slides  
- Global academic search integration for open-access resources  

---

### 🧠 AI Study Pavilion
- Automatic flashcard generation from lecture materials  
- Smart summaries for quick revision  
- AI tutor restricted to course content for accurate answers  

---

### 📅 Smart Academic Scheduling
- Department-wide class timetable system  
- Personalized student timetable  
- Real-time notifications before lectures  

---

### 📝 Past Questions Hub
- Organized exam questions by course and year  
- Easy revision and practice system
